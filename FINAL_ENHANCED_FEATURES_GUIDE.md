## 🚀 **How to Use New Features**

### **Access Your Enhanced Dashboard**
```bash
python clean_app.py
```
Open: `http://127.0.0.1:5002`

### **1. Setting Up Default Shares**
1. Go to **Master Lists** tab
2. Click **Edit** button next to any investor
3. Set **Default Shares** (e.g., 1, 2, 5 shares)
4. Save investor
5. Now when marking allotments, this number will be prefilled

### **2. Managing Master Data**
1. **Investors Master List**:
   - View all 32 imported investors
   - Edit names, UPI, banks, default shares
   - See family groupings clearly
   
2. **IPOs Master List**:
   - View all IPOs with buy/sale prices
   - Edit IPO details including sale prices
   - Track IPO status and profitability

### **3. Optimized IPO Management**
1. **Compact View**:
   - Quick overview of all IPOs
   - See key stats: applied count, allotted count, profit
   - Use "Expand" to drill down to specific IPO
   
2. **Bulk Operations**:
   - **Apply All Pending**: Apply for everyone across all IPOs
   - **Bulk Allotment**: Mark all applied as allotted
   - **Per-IPO Bulk**: Apply for all pending in one IPO

### **4. Enhanced Workflow Examples**

#### **Setting Up New IPO with Bulk Apply**
1. **Add IPO** → Master Lists → Add new IPO with details
2. **Compact View** → Switch to compact view for overview
3. **Bulk Apply** → Click "Bulk Apply" for the new IPO
4. **Switch to Detailed** → Change to detailed view for individual management

#### **Managing Allotments with Default Shares**
1. **Set Default Shares** → Master Lists → Edit investors → Set default shares
2. **Mark Allotments** → IPO-wise view → Click "Allot" 
3. **Auto-filled Shares** → Shares field shows default shares automatically
4. **Edit if Needed** → Click shares field to change if different

#### **Quick Profit Tracking**
1. **Set Sale Prices** → Master Lists → Edit IPOs → Add sale prices
2. **View Profits** → IPO-wise view → See real-time profit calculations
3. **Compact Overview** → Compact view → See total profit per IPO
